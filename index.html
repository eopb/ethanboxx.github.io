I was learning programming and electronics when I got sidetracked. I saw all of the awesome stuff some people had done in Blender3D and I wanted to be able to do it. I started to teach my self to model 3D objects using the internet and Youtube. The software I use most are Krita, Blender and GIMP. My pipeline is very open source.
